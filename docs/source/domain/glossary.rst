Glossary
========

.. glossary::

   stock-keeping unit
      Единица складского учета продукта (stock-keeping unit, SKU).

   customer
      Клиент

   order
      Заказ
   
   order reference
      Ссылка на заказ

   order line
      Строка товарной позиции

   quantity
      Количество товарных позиций

   batch
      Партия товара

   allocate order lines
      Разместить товарные позиции заказа в партиях товара

   available quantity
      Располагаемое количество товара в партии
   
   estimated arrival time
      Предполагаемый срок прибытия

   warehouse stock
      Товар на складе, товарный запас
